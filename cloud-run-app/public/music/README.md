# BGM について

このディレクトリの MP3 ファイルは **Docker ビルド時に自動生成** されます。
手動でファイルを用意する必要はありません。

## 生成の仕組み

`scripts/generate-bgm.sh` が ffmpeg の音声合成フィルターを使って
3 種類の BGM を生成します。

| ファイル | 和音 | 雰囲気 |
|---------|------|--------|
| bgm1.mp3 | Cメジャー | 明るい・爽やか |
| bgm2.mp3 | Fメジャー | 温かい・穏やか |
| bgm3.mp3 | Aマイナー | 落ち着き・やすらぎ |

各ファイル 30 秒、フェードイン/アウト付き。

## 手動で再生成する場合

```bash
cd cloud-run-app
bash scripts/generate-bgm.sh
```

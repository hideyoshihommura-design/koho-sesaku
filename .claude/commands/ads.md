# 広告コピー作成 Skill

Google広告・Meta広告（Facebook/Instagram）の広告テキストを作成します。
ターゲット・目的・フォーマットに合わせて複数パターンを提案します。

## 対応できる主なタスク

### 1. Google広告（検索広告）
- レスポンシブ検索広告（RSA）形式
  - 見出し（30文字以内）× 最大15パターン
  - 説明文（90文字以内）× 最大4パターン
- キーワードグループ別に作成
- 保存先: `google-ads/`

### 2. Google広告（ディスプレイ・P-MAX）
- ショートヘッドライン（30文字以内）
- ロングヘッドライン（90文字以内）
- 説明文（90文字以内）
- 画像・動画の構成案も提案

### 3. Meta広告（Facebook・Instagram）
- プライマリテキスト（本文）
- 見出し（40文字以内）
- 説明文（30文字以内）
- ターゲット別に複数パターン作成
- 保存先: `meta-ads/`

### 4. 広告戦略・設計
- ターゲット別の訴求軸整理
- A/Bテスト用パターン設計
- LP（ランディングページ）との整合性確認
- 入札戦略・予算配分の提案

---

## 実行手順

1. **広告の種別を確認**（Google検索 / Meta / ディスプレイ）
2. **目的・ターゲットを確認**（資料請求 / ウェビナー登録 / 認知拡大）
3. **既存の広告テキストを参照**して一貫性を保つ
   - `google-ads/` 内の既存ファイル
   - `meta-ads/` 内の既存ファイル
4. **複数パターン（最低3案）を作成**
5. **md形式でファイルに保存**

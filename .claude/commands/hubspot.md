# HubSpot CRM操作 Skill

HubSpot MCPを活用して、CRMデータの確認・操作・設定を行います。
ユーザーの依頼に応じて、HubSpot上のデータを検索・確認・更新します。

## 対応できる主なタスク

### 1. コンタクト・リード管理
- コンタクトの検索・一覧確認
- コンタクトのプロパティ確認・更新
- リードのステージ確認

### 2. ワークフロー確認
- 既存ワークフローの一覧・詳細確認
- ワークフローのエンロール状況確認
- メールシーケンスの配信状況確認

### 3. エンゲージメント・活動記録
- メール送信記録の確認
- ミーティング・コール記録の作成
- タスクの作成・確認

### 4. フォーム・プロパティ管理
- フォーム一覧・設定確認
- カスタムプロパティの確認・作成
- フォーム送信データの確認

### 5. レポート・分析支援
- オブジェクト（コンタクト・会社・取引）の件数確認
- セグメント別のリスト作成提案
- ナーチャリング進捗の確認

---

## 実行手順

1. **タスクを確認**（検索 / 確認 / 更新 / 作成）
2. **HubSpot MCPツールを使用**してデータを取得・操作
3. **結果をわかりやすく整理**してユーザーに報告
4. **次のアクション**を提案する

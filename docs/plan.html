やりたいこと
Google Drive の専用フォルダに素材（写真・テキストなど）を入れるだけで、
AI が各SNS用の投稿文・画像・リール動画を自動生成。
担当者が確認・承認したら、Facebook / Instagram / TikTok / X へ自動投稿。
投稿後は WordPress のお知らせに下書きを自動作成する。
全体フロー
①
素材を投入
担当者がGoogle Driveの専用フォルダに写真・コメント.txtを入れる
②
自動起動
毎朝9:00にCloud Schedulerがシステムを起動し、素材を取り込む
③
AI生成
ClaudeがSNS用投稿文を生成。VeoがInstagram・TikTok用リール動画を生成
④
確認通知
SlackまたはメールでAIが作ったコンテンツの確認依頼が届く
⑤
承認
確認画面で内容を確認・修正し、「承認」ボタンを押す
⑥
全SNS投稿
Facebook・Instagram・TikTok・X へ自動投稿される
⑦
WP下書き作成
WordPressのお知らせに下書きが自動作成。担当者が仕上げて公開

Google Drive 素材管理
①
フォルダ番号で順序管理
先頭に 001_ 002_ と番号をつける。小さい番号から順に処理。順序変更は番号を変えるだけ
②
素材は画像 ＋ コメント.txt
フォルダに画像（何枚でも可）とコメント.txtを入れる。コメントにAIへの説明・指示を自由に書く。画像なしでもOK
③
投稿済みは自動で移動
処理が完了したフォルダはシステムが「処理済み/」に自動移動。過去の投稿内容はそのまま参照できる

SNS別 投稿仕様
SNS API 投稿コンテンツ 文字数目安 備考
Facebook Meta Graph API 投稿文 ＋ 画像 〜300文字 ページ管理者権限が必要
Instagram Meta Graph API 投稿文 ＋ 画像 または リール動画 〜150文字 ＋ ハッシュタグ Facebookと同じAPIで可
TikTok TikTok Content Posting API リール動画 ＋ キャプション 〜100文字 ＋ ハッシュタグ Business API申請が必要（審査2〜4週間）
X（Twitter） X API v2 投稿文 ＋ 画像 〜140文字 有料プラン必須（Basic $100/月〜）

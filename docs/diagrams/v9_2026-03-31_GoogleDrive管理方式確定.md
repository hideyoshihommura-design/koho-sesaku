# v9 - 2026-03-31 - Google Drive 管理方式を確定

## 変更内容
- Google Drive フォルダ構成を確定
- 順序管理：フォルダ名の先頭番号（001_, 002_...）
- 素材構成：画像 ＋ コメント.txt（AIへの説明・指示）
- 投稿済み管理：`処理済み/` フォルダへ自動移動

## Google Drive フォルダ構成

```
投稿素材_キュー/
├── 001_スタッフ紹介/
│   ├── 写真.jpg
│   └── コメント.txt
├── 002_イベント告知/
│   ├── 写真1.jpg
│   ├── 写真2.jpg
│   └── コメント.txt
├── 003_施設紹介/
│   └── コメント.txt      ← 画像なしでもOK
└── 処理済み/             ← 投稿後にシステムが自動移動
    └── 001_スタッフ紹介/
```

## 3つのルール

| ルール | 内容 |
|--------|------|
| 順序 | フォルダ名の先頭に `001_` `002_` と番号をつける |
| 素材 | フォルダの中に画像（何枚でも可）と `コメント.txt` を入れる |
| 投稿済み | 処理が終わったフォルダはシステムが `処理済み/` に自動移動 |

## コメント.txt の書き方

```
今月入社したスタッフの山田さんの紹介です。
趣味はガーデニングで、利用者さんとの庭作りが得意。
明るくて元気な雰囲気が伝わるような投稿にしてください。
```
